# 🎲 Achtung! Cthulhu Dice Roller

A lightweight, private, and offline-ready dice roller specifically designed for Achtung! Cthulhu campaigns. No servers, no tracking—just fast, reliable rolls for your tabletop sessions.

## ✨ Features

- **Game-Specific Rolls**: Designed to handle the dice mechanics required for Achtung! Cthulhu.
- **100% Local**: Works entirely within your browser.
- **Offline Access**: Perfect for use at any game table, even without an internet connection.
- **PWA Ready**: Installable as a native app on mobile or desktop for quick access.
- **Privacy First**: No data collection; your rolls and settings stay on your device.

## 🚀 How to use

1. **Online**: [Open the live demo](https://gdr-sys.github.io/Achtung-Cthulhu-Dice-Roller/)
2. **Offline**: 
   - Clone this repository or download the `index.html`.
   - Open the file in any modern web browser.
   - Start rolling your dice!

## 🛠 Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Offline**: Service Workers (PWA)
- **Settings**: LocalStorage
